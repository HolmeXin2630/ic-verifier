# Requirements Template

Ask 3-5 at a time, wait for answers.

1. Component type? (UVC / agent / driver / monitor / scoreboard / SV package)
2. Protocol/interface?
3. Transaction — exists or create?
4. Usage? (standalone / SoC / reusable library)
5. Multi-instance?
6. Public API — how to instantiate?
7. Config knobs?
8. UVM phases? TLM ports?
9. Error handling?
10. Completion criteria? Simulator? (VCS / Xcelium / Questa)

## UVC-Specific
- Scoreboard? Coverage? Error scenarios? Backdoor?
